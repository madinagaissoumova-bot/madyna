# Mobile Checklist

## Navigation

- Menu button is obvious
- Open menu is readable and easy to dismiss
- Current page is visible
- Header actions remain tappable

## Layout

- No horizontal overflow
- Cards and sections breathe on small screens
- Text wraps cleanly
- Primary CTA stands out quickly

## Interaction

- Tap targets feel comfortable
- Buttons do not collide
- Forms use sensible input types and autofill
- Validation and helper text are easy to notice

## Conversion Flow

- The next step is obvious on cart and checkout
- Empty states guide the user forward
- Mobile summaries are scannable
- Confirmation language matches actual product behavior
